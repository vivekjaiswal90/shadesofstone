Other Configurations

1. Upload Banners

In this template, we offer some available banners for you to use conveniently. After the template is installed, banners may not be created. You can upload a new banner or edit it through our guide below

  • Login admin panel
  • From the top menu, go to Banners > Manage Banners.
  • Click button Add New Banner
  • Input some fields and browse to select banner
  • Click on Save Banner button

2. Add New Products

For new product to show in home page, you will need to select new for each product.

  • Login admin panel
  • From the top menu, go to Catalog > Manage Products.
  • Click to edit product you want to show.
  • Click on General tab on the left column
  • Choose Set Product as New from Date
  • Choose Set Product as New to Date
  • Click on Save button
