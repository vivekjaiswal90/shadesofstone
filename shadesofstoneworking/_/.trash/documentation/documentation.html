How To Install

1. Install template package

  • Step 1: Open www.buzzthemes.com, log in your account
  • Step 2: Go to My Account > My Downloadable Products and download template package
  • Step 3: Extract template package into your local computer. You can see some folders:
    • add-ons - Source code of extension
    • documentation - Documentation files
    • psd & font - Adobe Photoshop files & Fonts
    • sources - Magento theme files that should be uploaded to the root directory of your Magento site
      • 1.5.x
      • 1.6.x
      • 1.7.x
      • 1.8.x
    • theme_contents - Theme content (CMS pages & Static blocks) files
  • Step 4: Log into Magento backend and go to System > Cache Management, disable all cache type.
  • Step 5: Choose the most appropriate packages with your magento version from "sources" folder to upload to your site. After that, use FTP client (like Filezilla, WinSCP, cuteFTP) to upload (copy) folders to your Magento site root folder.

2. Template Configuration

Activate new template

Your web-site remains using the old template after this template has been installed. You can choose 1 in 2 solutions to activate new template

Solution 1:
  • Login admin panel
  • Go to System > Design
  • Click Add Design Change
  • Select the theme which you have created in folder structure
Solution 2:
  • Login admin panel
  • Go to System > Configuration
  • Choose tab Design on the left side menu
    • Package section: Fill the package name (Default: default)
    • Themes section: Set template for system by adding name of the new template.
    • Header section: Config new logo for system by changing name of logo image file. Get the name of logo in skin/frontend/package-name/template-name/images

During edit CMS and Static blocks, we recommend you to hide WYSIWYG editor by clicking on Show / Hide Editor button above the content section

CMS Pages Configuration

For almost template, you need to change the content of home page or add some specific pages.

Add & edit a CMS page:
  • Login admin panel
  • Select CMS > Pages from top menu
  • All pages will avaiable here. Open any of them to edit the page content. In case you need to add new page click the Add New Page button in the top right corner.
  • Input the Page Title and URL Key
  • From the left column select the Content tab. It contains the HTML code of the page. Get the content on txt file for each page from folder theme_contents/cms_pages
  • Switch to Design tab to change the layout of page and update XML content .

Click here to view list of CMS Pages.

During edit CMS and Static blocks, we recommend you to hide WYSIWYG editor by clicking on Show / Hide Editor button above the content section

Static Blocks Configuration

In the template, some layout objects will generate from a static block because they are not only show on home page. To add & edit the blocks to your store please use the instructions below.

Add & edit a static block:
  • Login admin panel
  • Select CMS > Static Blocks from top menu
  • Open a block on the list to edit the content.In case you want to add new block, click the Add New Block button in the top right corner
  • Input the Block Title and Identifier
  • Get the content on txt file for each block from folder theme_contents/static_blocks

Click here to view list of static blocks.
